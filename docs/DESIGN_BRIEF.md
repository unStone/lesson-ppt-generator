# Lesson PPT Generator — 产品设计文档（给设计师）

> 一个面向浙江小学数学教师的 AI 备课工作台，从教案文档一键生成精美 PPT。

---

## 一、产品定位

### 一句话
**"AI 帮老师把教案变成 PPT"** —— 上传教案文档，AI 自动分析、生成教学总览，确认后一键产出可编辑的演示文稿。

### 目标用户
- **核心用户**：浙江小学数学教师（1-6 年级）
- **高频场景**：日常备课、公开课准备、教研分享
- **痛点**：做 PPT 耗时费力、排版不专业、动画设计繁琐

### 核心差异化
不是通用 AI PPT 工具，而是**深度适配小学数学教学场景**的专用工具：
- 理解教案结构（教学目标、重难点、环节设计）
- 适配数学学科（公式、示意图、例题、思维导图）
- 符合小学课堂（大字体、色彩活泼、动画适度）

## 二、用户流程（4 步闭环）

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ ① 上传   │ → │ ② 总览   │ → │ ③ 生成   │ → │ ④ 预览   │
│  教案    │    │  编辑    │    │  等待    │    │  下载    │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### Step 1 — 上传教案（30 秒）
- 拖拽或点击上传 Word/TXT 教案文件
- 选择课型（日常课 / 公开课 / 示范课 / 比赛课 / 微课 / 教研课）
- 点击「开始解析」

### Step 2 — 总览编辑（1-3 分钟）
AI 解析教案后，自动生成结构化教学总览，老师可以：
- 确认/修改课题、年级、单元信息
- 查看教学目标（知识/能力/情感）
- 查看教学环节（导入→新授→练习→小结→作业）的时间分配
- 调整页数、主题风格、动画强度
- 确认无误后点击「生成 PPT」

### Step 3 — AI 生成（30-60 秒）
- 显示生成进度（解析 → 规划 → 设计 → 导出）
- 进度条可视化，有步骤提示

### Step 4 — 预览 & 下载
- 左侧：页面缩略图列表，可点击切换
- 右侧：大图预览，支持翻页
- 单页可编辑（文字、图片、布局微调）
- 底部：「下载 PPT」按钮

## 三、页面清单

### 页面 A — 首页 / 上传页
**功能**：文件上传 + 课型选择

**关键元素**：
- 顶部标题栏：Logo + 产品名 + 版本号
- 主体：大号拖拽区域（虚线框，带文件图标）
- 课型选择：6 个卡片网格（日常/公开/示范/比赛/微课/教研）
- 底部：「开始解析」主按钮（上传后才可点击）

**参考风格**：Dropbox 上传页 / Figma 导入页 —— 简洁、留白、操作聚焦

### 页面 B — 总览编辑页
**功能**：展示 AI 解析结果，支持人工审核修改

**关键元素**：
- 顶部：课题名 + 年级 + 单元（卡片式）
- 教学目标区：三个维度（知识/能力/情感），可编辑
- 重难点区：左右两栏（重点/难点），高亮色块区分
- 教学环节区：横向流程图（导入→新授→练习→小结→作业），显示每环节页数和时间
- 设置区：总页数 / 主题风格 / 动画强度
- 底部操作栏：「确认并生成 PPT」主按钮 + 「重新解析」次按钮

**参考风格**：Notion 数据库视图 + Airtable 卡片 —— 结构化、可编辑、信息分层清晰

### 页面 C — 生成中页
**功能**：展示 AI 工作进度

**关键元素**：
- 居中：环形/条形进度条
- 下方：6 个步骤节点（解析→总览→内容→素材→动画→导出），逐个高亮
- 提示文案："AI 正在设计第 3 页内容..."

**参考风格**：线性进度指示器，类似 Figma 导出 / Vercel 部署

### 页面 D — 预览页
**功能**：PPT 预览 + 单页编辑 + 下载

**关键元素**：
- 左侧边栏：页面缩略图垂直列表（类似 PowerPoint 左侧导航）
- 主区域：当前页大图预览（16:9 比例，白色背景容器）
- 顶部工具栏：当前页码 / 「编辑此页」/ 「下载 PPT」
- 底部：翻页箭头 + 页码指示器

**参考风格**：Google Slides / Canva 预览模式

### 页面 E — 单页编辑器（弹窗/侧滑）
**功能**：微调单页内容

**关键元素**：
- 标题输入框
- 内容文本域（支持多行）
- 布局模板选择（4 种缩略图）
- 图片替换按钮
- 保存 / 取消

**参考风格**：Canva 右侧属性面板

### 页面 F — 设置面板（弹窗）
**功能**：AI 模型配置

**关键元素**：
- API Key 输入（密码框）
- 模型选择下拉框（GPT-4o / GPT-4o-mini / Claude 等）
- Provider 选择（OpenAI / Anthropic / 本地 Ollama）
- 保存按钮

## 四、设计原则

### 1. 教师友好
- **大字号**：正文 ≥ 14px，标题 ≥ 20px，教室投影场景需要清晰可读
- **高对比度**：避免浅色文字在浅色背景上
- **少层级**：核心操作不超过 2 步可达

### 2. 专业可信
- **蓝色主色调**：教育行业信任色（参考钉钉、企业微信）
- **卡片式布局**：信息模块化，边界清晰
- **状态明确**：按钮状态（默认/悬停/禁用/加载）区分明显

### 3. 活泼适度
- **圆角卡片**：8-12px 圆角，柔和不僵硬
- **微动画**： hover 效果、进度条过渡，避免过度花哨
- **数学元素点缀**：几何图形、数字、公式符号作为装饰元素

### 4. 深色/浅色模式
- 默认浅色模式
- 预留深色模式设计空间（后续迭代）

## 六、字体建议

- **标题**：系统默认无衬线（PingFang SC / Microsoft YaHei / Segoe UI）
- **正文**：同上
- **数字/公式**：等宽字体 fallback（Courier / Consolas）
- **英文**：Inter / SF Pro

---

## 七、布局规范

- **窗口尺寸**：默认 1400×900，最小 1200×700
- **顶部导航栏**：高度 56px，白色背景 + 底部 1px 边框
- **内容区**：左右 padding 24px，最大内容宽度 1200px（居中）
- **卡片间距**：16px
- **圆角**：卡片 12px，按钮 8px，输入框 6px

---

## 八、动效建议

| 场景 | 效果 | 时长 |
|------|------|------|
| 页面切换 | 淡入淡出 | 200ms |
| 卡片 hover | 轻微上移 + 阴影加深 | 150ms |
| 进度条 | 平滑过渡 | 300ms |
| 文件拖拽 | 边框变蓝 + 背景变浅蓝 | 即时 |
| 生成完成 | 绿色勾号弹出 | 300ms |
